# Ascension

Ascension est un framework Front-end javascript. Il contient plusieurs fonctions permettant de faciliter la création de composant natif.

## Documentation

Vous pouvez en lire plus en regardant la documentation.

## Installer

Pour installer la librairie, copiez simplement l'un des deux fichiers se situant dans le dossier dist et importez le fichier.
```html 
<script src="ascension-min.js"></script>
```

## Modifier le code

Vous pouvez librement modifier le code. Le point d'entrée est le fichier index.js sur la racine du repository.
Utilisez la commande `npm run build` pour recréer la librairie en utilisant rollup.
